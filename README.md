# TaskMaster Pro - Task Management Web Application

## Oasis Infobyte Internship

### Level 2 - Task 3

## Project Overview

TaskMaster Pro is a responsive task management web application developed using HTML, CSS, and JavaScript.

The application helps users efficiently manage daily tasks through features such as task creation, editing, deletion, completion tracking, priority management, due-date scheduling, search functionality, dark mode, and local storage persistence.



## Features

### Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

### Organization Features
- Due Date support
- Priority Levels
  - High
  - Medium
  - Low
- Search Tasks

### Dashboard
- Pending Task Counter
- Completed Task Counter

### User Experience
- Dark Mode / Light Mode
- Responsive Design
- Enter Key Support

### Data Persistence
- Local Storage Integration
- Tasks remain available after browser refresh
